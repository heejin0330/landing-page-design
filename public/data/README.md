# 학교 데이터 파일 안내

## 파일 위치
이 폴더에 공공데이터포털에서 다운로드한 엑셀 파일을 저장하세요.

## 파일명
`school-list.xlsx`

## 다운로드 방법

1. 공공데이터포털 접속
   - URL: https://www.data.go.kr/data/15118999/fileData.do

2. "파일 다운로드" 버튼 클릭

3. 다운로드한 엑셀 파일을 이 폴더에 `school-list.xlsx`로 저장

## 엑셀 파일 구조

엑셀 파일에는 다음 컬럼이 포함되어 있어야 합니다:
- 학교명 (또는 학교명(한글), 학교명(영문))
- 학교유형 (또는 유형)
- 시도 (또는 시/도, 지역)
- 시군구 (또는 시/군/구, 구/군) - 선택사항
- 교육청 (또는 소속교육청) - 선택사항
- 학과 (또는 과, 전공)

## 참고사항

- 파일이 없으면 학교 검색 기능이 작동하지 않습니다.
- 엑셀 파일의 컬럼명이 다를 경우 `actions/school.ts` 파일의 `loadSchoolDataFromExcel` 함수를 수정해야 합니다.
